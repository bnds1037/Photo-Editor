# Photo Editor - macOS照片管理与编辑应用

## 项目介绍

Photo Editor是一款专为macOS设计的现代化照片管理与编辑应用，采用SwiftUI框架开发，遵循MVVM架构模式。应用集成了macOS 26最新的Liquid Glass（毛玻璃）设计语言，提供流畅的用户体验和强大的照片管理功能。

## 主要功能

### 📸 照片管理
- **照片浏览**：支持网格视图和时间线视图两种浏览模式
- **照片搜索**：基于标签和位置的智能搜索功能
- **照片导入**：支持从本地文件夹导入照片
- **照片删除**：支持单个和批量删除照片
- **照片移动**：支持将照片移动到不同相册

### 📁 相册管理
- **创建相册**：自定义相册名称和描述
- **相册浏览**：查看相册中的所有照片
- **相册编辑**：修改相册名称和描述
- **相册删除**：删除不需要的相册
- **智能分类**：自动按时间、地点分类照片

### 🎨 照片编辑
- **基础调整**：亮度、对比度、饱和度调节
- **滤镜效果**：多种预设滤镜效果
- **裁剪旋转**：照片裁剪和旋转功能
- **色彩校正**：色温、色调调节
- **实时预览**：编辑效果实时预览

### 🤖 智能功能
- **人脸识别**：自动检测照片中的人脸
- **场景分类**：智能识别照片场景（风景、人物、建筑等）
- **模糊检测**：自动检测模糊照片
- **重复检测**：识别重复或相似照片
- **智能搜索**：基于内容的智能搜索

### ⚡ 批量操作
- **批量删除**：一次性删除多张照片
- **批量移动**：将多张照片移动到指定相册
- **批量编辑**：批量应用滤镜和调整
- **批量导出**：批量导出照片到本地

### 🌟 Moment功能
- **精选展示**：随机展示收藏夹中的照片
- **轮播播放**：自动轮播精选照片
- **桌面小组件**：支持桌面小组件显示

### 🎯 桌面小组件
- **小尺寸小组件**：显示单张照片和快捷操作
- **中等尺寸小组件**：显示多张照片和操作按钮
- **大尺寸小组件**：显示照片网格和完整操作
- **实时更新**：小组件内容实时更新

## 技术架构

### 架构模式
- **MVVM架构**：Model-View-ViewModel分层架构
- **响应式编程**：使用Combine框架实现数据绑定
- **依赖注入**：通过EnvironmentObject实现依赖注入

### 技术栈
- **SwiftUI**：现代化声明式UI框架
- **AppKit**：macOS原生框架
- **CoreData**：数据持久化存储
- **Vision**：机器学习和图像识别
- **Photos**：照片库访问和管理
- **WidgetKit**：桌面小组件开发
- **Combine**：响应式编程框架

## 安装和运行

### 系统要求
- macOS 12.0 或更高版本
- Xcode 14.0 或更高版本
- Swift 5.0 或更高版本

### 安装步骤

1. **克隆项目**
   ```bash
   git clone [项目地址]
   cd "Photo Editor"
   ```

2. **打开项目**
   ```bash
   open "Photo Editor.xcodeproj"
   ```

3. **配置签名**
   - 在Xcode中选择项目
   - 在"Signing & Capabilities"标签页中选择你的开发团队

4. **运行应用**
   - 选择"My Mac"作为目标设备
   - 点击运行按钮或按`Command+R`

### 权限配置

应用需要以下权限：
- **照片库访问**：读取和管理照片
- **摄像头访问**：拍摄新照片
- **位置访问**：获取照片位置信息

在首次运行时，应用会请求这些权限。

## 功能详解

### 照片浏览模式

#### 网格视图
- 自适应网格布局
- 支持缩放调整
- 快速预览和选择

#### 时间线视图
- 按时间排序显示
- 显示照片详细信息
- 支持位置标签显示

### 智能搜索

应用支持多种搜索方式：
- **关键词搜索**：基于标签和描述
- **位置搜索**：基于拍摄地点
- **时间搜索**：基于拍摄时间
- **智能搜索**：基于图像内容识别

### 照片编辑工具

#### 基础调整
- 亮度：-100 到 +100
- 对比度：-100 到 +100
- 饱和度：-100 到 +100

#### 高级功能
- 色温调节
- 色调调节
- 锐化处理
- 降噪处理

### 桌面小组件

小组件支持三种尺寸：
- **小尺寸（2x2）**：显示单张照片和快捷按钮
- **中等尺寸（4x2）**：显示两张照片和操作按钮
- **大尺寸（4x4）**：显示照片网格和完整操作
